Démonstration de d3.js pour les aspects suivants:

acquisition et traitement d'un json
transformation d'une échelle de dates en échelle numérique
mise à l'échelle automatique

visualisation d'un nuage de points
attribution d'un identificateur spécifique, permettant de jeter des points obsolètes
filtrage numérique de données temporelles et visualisation de l'effet

le html est minimal en ce qu'il s'agit d'une démo de faisabilité
le code est dans le html par souci de compacité

d3.js n'est utilisé que pour la visualisation,
jQuery pour toutes les autres fonctionnalités
